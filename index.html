Mon club

Utilisé pour les matchs (résultats, lieu domicile/extérieur).

Équipes

Gérez jusqu'à plusieurs équipes (ex: U15, U17...). Chaque équipe a son propre effectif et ses propres séances.

Code d'ouverture

Protégez l'application avec un code. Laissez vide pour ne pas en utiliser.

Sauvegarde des données

Les données restent sur cet appareil uniquement. Pour en garder une copie en dehors du téléphone, exportez-les régulièrement dans un fichier, puis enregistrez-le où vous voulez via le menu de partage (app Fichiers, e-mail, etc.).

Sauvegardes automatiques sur l'appareil

L'application conserve automatiquement les 3 dernières sauvegardes (une par jour d'utilisation). À chaque nouvelle sauvegarde, la plus ancienne est effacée. Elles restent sur cet appareil : pour une copie en dehors du téléphone, utilisez « Exporter les données ».

Aucune sauvegarde automatique pour l'instant.

Synthèse Excel

Génère un fichier Excel (.xlsx) avec toutes les statistiques de l'équipe active (présences, temps de jeu, buts, cartons) et la liste des matchs. Nécessite d'avoir ouvert l'app au moins une fois avec internet.

À propos des données

Toutes les données (joueurs, séances, présences) sont enregistrées uniquement sur cet appareil, dans ce navigateur. Rien n'est envoyé sur internet ni partagé. Si vous effacez les données de Safari pour ce site ou changez d'appareil, ces données seront perdues — pensez à exporter régulièrement si besoin.
